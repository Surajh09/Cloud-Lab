Merge Sort

Description Reviews Details

Merge sort is a sorting algorithm that uses a divide-and-conquer approach. It recursively divides an array into two halves until there is only one element in each half. It then merges the two halves by comparing and sorting them, creating a new sorted array. This process is repeated until the entire array is sorted.

Time Complexity O(n(log n))
Stability Stable
